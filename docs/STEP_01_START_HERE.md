# Step 1 - Start Here

You are not behind. We are going to do this one clean step at a time.

## Goal of this zip

This folder gives you a clean project structure so we stop working from random notes and start working from a real project.

## Your next move

Install these:

1. **Node.js LTS**
2. **VS Code**
3. **pnpm** (later, if needed)

## Folder guide

### `apps/web`
This will become the web dashboard and live event page.

### `apps/bot`
This will become the Discord bot.

### `apps/worker`
This will run reminders, timers, and background jobs.

### `packages/db`
This will hold the database schema and migrations.

### `packages/core`
This will hold the custom Development Division format logic.

### `packages/discord`
This will hold Discord message builders and interaction helpers.

### `packages/config`
This will hold shared settings and environment validation.

### `packages/ui`
This will hold reusable web UI pieces.

## Important

This is just the scaffold.
The next real build step should be one of these:

- set up Node + VS Code
- create the first Discord bot file
- create the database schema
