# Development Division Tournament System

This is a starter scaffold for a Discord-first tournament platform for **THE FINALS – Development Division**.

## What is included

- Monorepo folder structure
- Placeholder apps for:
  - `apps/web`
  - `apps/bot`
  - `apps/worker`
- Shared packages for:
  - `packages/db`
  - `packages/core`
  - `packages/discord`
  - `packages/config`
  - `packages/ui`
- A simple step-by-step setup guide

## What is NOT included yet

This zip is a **starting structure**, not a finished bot.
It does **not** yet include:

- working Discord bot commands
- database schema
- web dashboard pages
- Google Forms import
- live tournament logic

We will build those one piece at a time.

## Recommended first steps

1. Unzip the folder
2. Open it in VS Code
3. Read `docs/STEP_01_START_HERE.md`
4. Install Node.js LTS
5. Come back and we build the next piece together
